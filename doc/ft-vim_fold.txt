INTRODUCTION					*ft-vim_fold-introduction*

*ft-vim_fold.vim* is a Vim plugin to provide the setting for folding a Vim
script file.

This plugin folds:
- Between `function` / `endfunction`
- Between `augroup xxx` / `augroup END`
- Line continuation
- 'foldmarker' option


Requirements:
- Vim 8.1 or later

Latest version:
https://github.com/thinca/vim-ft-vim_fold
